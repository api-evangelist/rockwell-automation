# Rockwell Automation

Rockwell Automation is a global provider of industrial automation and digital transformation solutions, helping manufacturers boost productivity, sustainability, and agility. The company offers a portfolio of hardware, software, and services including programmable logic controllers (PLCs), industrial networking, motor control, safety systems, and the FactoryTalk software suite for MES, SCADA, historian, remote access, and industrial data management.

**Website:** [https://www.rockwellautomation.com](https://www.rockwellautomation.com)

**Documentation:** [https://www.rockwellautomation.com/en-us/support/documentation.html](https://www.rockwellautomation.com/en-us/support/documentation.html)

**Developer Portal:** [https://developers.plex.com/docs](https://developers.plex.com/docs)

**GitHub:** [https://github.com/RockwellAutomation](https://github.com/RockwellAutomation)

## APIs

### Plex ERP API
Plex by Rockwell Automation provides a Smart Manufacturing Platform ERP API with REST/JSON endpoints for customer orders, shipping, production, quality, and just-in-sequence parts delivery.

- **Documentation:** [https://plex.rockwellautomation.com/en-us/resources/enterprise-resource-planning-erp-apis.html](https://plex.rockwellautomation.com/en-us/resources/enterprise-resource-planning-erp-apis.html)
- **Portal:** [https://developers.plex.com/docs](https://developers.plex.com/docs)

### FactoryTalk DataMosaix API
Industrial DataOps platform providing REST API access for industrial data management, contextualization, and analytics.

- **Documentation:** [https://www.rockwellautomation.com/en-us/docs/factorytalk-datamosaix/current/online-help-ditamap/api-reference.html](https://www.rockwellautomation.com/en-us/docs/factorytalk-datamosaix/current/online-help-ditamap/api-reference.html)

### FactoryTalk Remote Access Web API
RESTful HTTP interface for remote access management of industrial devices.

- **Documentation:** [https://www.rockwellautomation.com/en-pr/docs/factorytalk-remote-access/ubiquity-ftra-help/ftra-ditamap/web-api.html](https://www.rockwellautomation.com/en-pr/docs/factorytalk-remote-access/ubiquity-ftra-help/ftra-ditamap/web-api.html)

### Logix Designer SDK
Studio 5000 Logix Designer SDK for CI/CD pipeline automation and version control of PLC programs.

- **GitHub:** [https://github.com/RockwellAutomation/ra-logix-cicd](https://github.com/RockwellAutomation/ra-logix-cicd)

### Emulate3D API
3D simulation and digital twin API for virtual commissioning and factory automation.

- **GitHub:** [https://github.com/RockwellAutomation/ra-emulate3d-core-samples](https://github.com/RockwellAutomation/ra-emulate3d-core-samples)
